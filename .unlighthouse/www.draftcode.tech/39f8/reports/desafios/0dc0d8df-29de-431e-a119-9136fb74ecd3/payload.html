    HTML
    CSS

    Esta página de login ira consolidar seu conhecimento em HTML e CSS será excelente para testas suas habilidades responsivas.

    Iniciante

    Requisitos

    O objetivo deste desafio é fazer uma tela de login, onde ira consolidar seus conhecimentos em html, csss e responsividade, busque deixar o mais próximo possível do design. Você pode usar qualquer ferramenta que desejar para ajudá-lo a concluir o desafio. se você tem algo que gostaria de praticar, sinta-se à vontade para tentar. O usuário deve ser capaz de: Visualize o layout ideal dependendo do tamanho da tela do dispositivo recomendamos trabalhar focando nas seguintes telas: Desktop: 1440x1024px Laptop: 1024x592px Mesa digitalizadora: 768x592px Mobile: 320x522px Espero que gostem do desafio, fiquem a vontade para adicionar comentário, estamos em fase de desenvolvimento
